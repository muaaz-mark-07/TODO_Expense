TODO_Expense
⚡ FlowDesk — A sleek, dark-themed Todo List + Expense Tracker built with pure HTML, CSS &amp; JavaScript. Features task priorities, categories, progress tracking, financial analytics, and localStorage persistence.
⚡ FlowDesk — Task & Finance Manager

A production-grade, single-file frontend project** combining a smart To-Do List with a personal Expense Tracker. Built entirely with **vanilla HTML, CSS, and JavaScript** — no frameworks, no dependencies.

> Built as a portfolio project to demonstrate frontend development skills.


 🚀 Live Demo

> Open `todo-expense-tracker.html` directly in any browser — no build step needed.



 ✨ Features
✅ Task Manager
- Add tasks with **priority levels** (High / Medium / Low)
- Assign **categories** (Work, Study, Personal, Health, Other)
- **Filter tasks** by status (All / Active / Done) or priority
- Live **progress bar** with completion percentage
- Stats dashboard — total, completed, pending, completion rate
- **Persistent storage** using `localStorage`

💰 Expense Tracker
- Log **income & expense** transactions
- Assign **categories** (Food, Transport, Shopping, Health, etc.)
- Real-time **net balance** with positive/negative color feedback
- **Category breakdown** with animated bar chart
- **7-day daily spending** mini bar chart
- Filter transactions by type



 🛠️ Tech Stack
